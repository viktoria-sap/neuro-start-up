# NeuroStartUp

**NeuroStartUp** - динамически развивающийся стартап, специализирующийся на поиске с использованием новейших технологий искусственного интеллекта. 
Наши преимущества:
* Высокая точность поиска
* Высокая скорость поиска
* Низкая цена

## Начало работы

Чтобы скопировать данный репозиторий на *локальную машину*, необходимо:
1. Нажать на зеленую кнопку "Clone or download" и скопировать URL нужного репозитория.
1. Открыть git в рабочей папке правой кнопкой мыши с помощью команды Git Bash Here (для Windows).
1. Дать команду 'git clone' и скопированный ранее URL нужного репозитория.
